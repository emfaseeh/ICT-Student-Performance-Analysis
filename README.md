AICT Open-Ended Lab Project
Title: AI-Integrated Student Performance Analysis

Course: Application of ICT (AICT)
Lab Type: Open-Ended Lab

Tools Used:
- MS Excel (Formulas, Conditional Formatting, Macros)
- Power BI (Dashboard & Visualization)
- AI Tools (ChatGPT, Grammarly)
- Collaboration Platforms (Google Drive, GitHub)

Description:
This project focuses on analyzing student academic performance
using ICT and AI-assisted tools. Excel was used for data processing,
automation, and conditional formatting. Power BI was used to create
interactive dashboards and visual insights. AI tools were used for
documentation improvement and structuring.

Collaboration:
Google Drive was used for file sharing and teamwork.
GitHub was used to maintain datasets and documentation.
